# 马克思主义基本原理复习考试系统

基于 Python Tkinter 开发的马克思主义基本原理复习考试系统，提供图形化界面，支持单选题、多选题和判断题三种题型的练习和考试。

## 📁 项目结构

```
马原/
├── quiz.py                    # 命令行版本题库程序
├── quiz_gui.py                # 图形界面版本题库程序（推荐使用）
├── 马克思主义基本原理题库.txt     # 题库数据源（txt格式）
├── 马克思主义基本原理2026.7复习.files/  # 旧版Word文档资源文件夹（保留）
└── readme.md                  # 项目说明文档
```

## 🚀 快速开始

### 环境要求

- Python 3.6+
- tkinter 库（Python 内置，一般无需额外安装）

### 运行方式

**图形界面版**：

```bash
python quiz_gui.py
```

## 🎮 使用说明

### 首页界面

1. **加载题库**：点击"加载题库"按钮，系统会自动读取题库文件并显示统计信息
2. **查看统计**：加载成功后显示各题型数量和本次考试配置
3. **开始考试**：点击"开始考试"按钮进入答题界面

### 答题界面

1. **题目信息**：顶部显示当前题目进度（如"第 1 / 80 题"）和题目类型
2. **选择答案**：
   - 单选题/判断题：点击单选按钮选择
   - 多选题：点击复选框选择多个选项
3. **提交答案**：选择答案后点击"提交"按钮验证正确性
4. **题目导航**：
   - 点击"上一题"返回上一道题
   - 点击"下一题"跳转到下一道题
5. **退出答题**：点击右上角"退出"按钮退出考试（会弹出确认对话框）

### 结果界面

1. **成绩统计**：显示总分、正确数、错误数和正确率
2. **评价等级**：
   - 80%以上：🎉 优秀！继续保持！
   - 60%-80%：👍 良好！还需努力！
   - 60%以下：💪 加油！多做练习！
3. **重新考试**：点击"再考一次"按钮重新随机出题
4. **返回首页**：点击"返回首页"按钮回到主界面

## ⚠️ 注意事项

1. 确保 `马克思主义基本原理题库.txt` 文件与程序在同一目录下
2. 答题时需先选择答案，"提交"按钮才会变为可用
3. 只有点击"提交"后答案才会被保存和验证
4. 点击"上一题"/"下一题"才会跳转题目
5. 退出考试后答题进度将丢失，请谨慎操作
6. 多选题答案顺序不敏感（如"AB"和"BA"视为相同答案）

## ❓ 常见问题

**Q: 运行时提示找不到文件？**

A: 请确保所有文件都在同一个目录下，特别是 `马克思主义基本原理题库.txt` 文件必须存在。

**Q: 判断题选择后没有反应？**

A: 选择答案后需要点击"提交"按钮才能验证答案。

**Q: 多选题如何选择多个选项？**

A: 使用鼠标点击多个复选框即可，点击"提交"按钮验证答案。

**Q: 题库文件是什么格式？**

A: 题库文件是txt格式，编码为UTF-8，包含题目、选项和答案信息。

## 📝 题库文件格式说明

```
一.选择题

A、 选项A内容

二.判断题
1. 判断题目内容。( )
答案：正确

```
